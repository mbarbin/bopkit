Module Bopkit_simulator

Once a bopkit project has been converted to a circuit, this module allows to simulate its execution. By default, the simulation runs in a loop and can be interrupted with a SigInt. You can also specify via the Config.t a finite number of cycles.

Inputs are read from stdin, and outputs produced on stdout, using the chars '0' and '1' to encode for signal values.
