Module Partial_bit_array

Sometimes some of the bits of a given computation are not specified (sometimes called "outputs with bit don't care"). This modules allows to represent such cases.

In contexts where bits don't care are encoded as human readable characters, we use the star char '*' to designate an unspecified bit.
