Module Bopkit_circuit.Circuit

A bopkit circuit holds a Cds.t along with information required to fire the computation of gates that are not entirely combinatorial, such as memories and external nodes.

The type is exposed because it's really just a bare record of useful information, and used by bop2c and the bopkit simulator.

The simulation logic is in lib/bopkit_simulator.

type t = private {
  1. path : Fpath.t;
  2. main : string;
  3. rom_memories : Bit_matrix.t array;
  4. external_blocks : Bopkit.Expanded_netlist.external_block array;
  5. cds : Cds.t;
  6. input_names : string array;
  7. output_names : string array;
}
val create_exn : path:Fpath.t -> main:string -> rom_memories:Bit_matrix.t array -> external_blocks:Bopkit.Expanded_netlist.external_block array -> cds:Cds.t -> input_names:string array -> output_names:string array -> t
