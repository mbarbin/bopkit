Library bopkit.bit-utils

This library exposes the following toplevel modules:

  • Bit_array In bopkit, bool arrays have many uses, including saving or reading the contents of memory-units from files, exchanging the value of buses between external blocks, printing block outputs to stdout, etc.
  • Bit_counter
  • Bit_matrix Extending Bit_array for arrays of 2 dimensions. For examples, loading/saving contents of memories.
  • Bit_string_encoding Some bit encoding that are used throughout the project.
  • Partial_bit
  • Partial_bit_array Sometimes some of the bits of a given computation are not specified (sometimes called "outputs with bit don't care"). This modules allows to represent such cases.
  • Partial_bit_matrix Extending Partial_bit_array for arrays of 2 dimensions.
